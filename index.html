Overcoat.js is a jQuery plugin
that provides simple labeled overlays
that respond to browser resize.

Step 1.

Add data-overcoat="name" and data-overcoat-label="Hello!" to any element. 

              <div data-overcoat="pink"
     data-overcoat-label="Pink's Label">
</div>

Step 2.

Add data-overcoat-toggle="name" to any clickable element. This element will now toggle the overcoat on click.

Step 3.

To initialize the overcoats on page, add the following:

              $("[data-overcoat-toggle]").overcoat();

Overcoat.js

Overcoat.js is a jQuery plugin that adds custom overlays to portions of a site. Data attributes are used to configure overcoat. See below for the list of options.

One of the key features of Overcoat.js is the ability for overcoats to resize as the browser windows resizes. Bacon.js is used to throttle how often overcoat responds to window resize events to avoid performance issues.

data attributes

name description example
overcoat Overcoat identifier data-overcoat="name"
overcoat-label Text displayed with the overcoat data-overcoat-label="Hello World!"
overcoat-bgcolor Overcoat background color. Any valid CSS color will work. data-overcoat-bgcolor="rgba(125, 211, 201, 0.6)"
overcoat-bgcolor Overcoat text color. Any valid CSS color will work. data-overcoat-color="rgba(125, 211, 201, 0.6)"

Contribute!

Overcoat.js was born out of a "How would you do it?" clone of Purplecoat.js. and an excuse to try out Bacon.js. We would love to see any improvements!
